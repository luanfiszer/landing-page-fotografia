## ✨ Visão Geral

O **Rio Solar Vibe** não é apenas um portfólio; é uma experiência digital imersiva construída para fotógrafos que capturam emoção e luz. Utilizando tecnologias web de ponta, a landing page proporciona uma jornada fluida e de alta performance através das lentes de um profissional baseado no Rio de Janeiro.

### 🌅 O Diferencial: "Golden Hour"
Nosso hook customizado `useGoldenHour` ajusta dinamicamente o tema e as animações com base na posição do sol, criando uma atmosfera única enquanto você navega.

## 🚀 Recursos

-   **🌊 Scroll Ultra-Suave**: Potencializado pelo **Lenis** para uma sensação de navegação cinematográfica.
-   **🪄 Animações de Alto Nível**: Uma mistura de **GSAP ScrollTrigger** e **Framer Motion** para micro-interações e storytelling baseado em scroll.
-   **🖼️ Portfólio Dinâmico**: Um álbum de fotos otimizado e responsivo usando `react-photo-album`.
-   **✨ Cursor Interativo Customizado**: Um cursor exclusivo que reage a elementos interativos.
-   **📐 Excelência Tipográfica**: Animações de texto via `split-type` para um visual editorial premium.
-   **📱 Totalmente Responsivo**: Perfeito em tudo, desde dispositivos móveis até telas ultra-wide.

## 🛠️ Tecnologias

### Core Frontend
-   **React 19**: Utilizando as últimas funcionalidades e otimizações de performance do React.
-   **Vite**: Para um ambiente de desenvolvimento ultra-rápido e builds otimizados.
-   **Tailwind CSS 4**: Um workflow moderno focado em CSS com alta utilidade.

### Animação e Interação
-   **GSAP + ScrollTrigger**: Animações avançadas atreladas ao scroll.
-   **Framer Motion**: Transições suaves de componentes e feedback de UI.
-   **Lenis**: O padrão da indústria para scroll suave.

### Componentes e Ferramentas
-   **Lucide React**: Iconografia limpa e consistente.
-   **Split Type**: Manipulações de texto em nível de caractere e palavra.
-   **React Photo Album**: Layouts de imagem flexíveis e performáticos.

## ⚡ Início Rápido

### ⚙️ Pré-requisitos
-   [Node.js](https://nodejs.org/) (v18.0.0 ou superior)
-   [npm](https://www.npmjs.com/) ou [pnpm](https://pnpm.io/)

### 🛠️ Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/landing-page-fotografia.git

# Navegue até o diretório
cd landing-page-fotografia

# Instale as dependências
npm install
```

### 🚀 Rodando Localmente

```bash
# Inicie o servidor de desenvolvimento
npm run dev
```
